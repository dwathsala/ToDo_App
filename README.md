# ✅ ToDo App (Python Tkinter)

A simple and user-friendly ToDo application built using Python and Tkinter.  
This application allows users to manage daily tasks with a graphical interface and ensures data persistence using a text file.

## 🚀 Features

- Add new tasks easily by typing and pressing Enter  
- View all tasks in a list format  
- Delete selected tasks with a single click  
- Automatic saving of tasks (persistent storage)  
- Tasks are loaded automatically when the app starts  

## 🛠️ Tech Stack

- Python 3  
- Tkinter (GUI Library)  
- File Handling (Text File Storage)

## 📂 Project Structure

- `app.py` → Main application script  
- `tasks.txt` → Stores user tasks  
- `check.ico` → Application icon
